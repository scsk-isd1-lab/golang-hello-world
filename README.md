# Golang Hello World

シンプルなGo言語の「Hello World」コンソールアプリケーション

## 説明

このプロジェクトは、標準出力に「Hello World」というテキストを表示するGo言語のシンプルなコンソールアプリケーションです。

## 必要条件

- Go 1.13以上

## インストールと実行方法

```bash
# リポジトリのクローン
git clone https://github.com/scsk-isd1-lab/golang-hello-world.git

# プロジェクトディレクトリへ移動
cd golang-hello-world

# アプリケーションの実行
go run main.go
```

## 出力例

```
Hello World
```
